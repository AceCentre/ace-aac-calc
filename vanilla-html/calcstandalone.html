Calculator Help

Keyboard Shortcuts: Enter - Evaluate expression Escape - Clear workpad ↑ - Previous expression ↓ - Next expression Alt/⌘ + D - Toggle Degrees/Radians Alt/⌘ + P - Toggle Decimal Places Ctrl + (0-9) - Set decimal places Ctrl/⌘ + C - Copy highlighted expression Memory Operations: Ctrl/⌘ + P - Memory Plus (M+) Ctrl/⌘ + M - Memory Minus (M-) Ctrl/⌘ + R - Memory Recall (MR)
